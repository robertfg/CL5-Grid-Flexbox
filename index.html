CSS Grid and Flexbox Examples

Flexbox Example No.1

1D layout - when you wrap flex items (flex-wrap), each new row becomes a flex container.
Space is distributed evenly across the row.
Use Flexbox when you want to space items evenly in a container; works from content out.
With the flex layout, if we drag our window wider and smaller, the flexbox nicely deals with adjusting the number of items in each row according to the available space. If we have a lot of space all five items can fit on one row, if we have a very narrow container we might just have space for one.

CSS Grid Example No.5

Grid and absolutely positioned elements

Make the grid container a containing block, add the position property to the container with a value of relative.
Making the position relative changes the positioning context for the items.
Give gbox5-3 a position value of absolute. Its position is now relative to the parent container, not the viewport.
The box will always be 40px to the right of grid-column 2 and 40px below grid-row 1.

CSS Grid Example No.7

Grid and display: contents

Set an item to display: contents and the box it would normally create disappears.
Boxes of child elements appear as though they've risen a level.
Sub-items become grid items and lay themselves out using auto-placement rules.
Get items nested into the grid to act as if they're part of the grid.
This doesn't work in Atom's preview. You'll need to display it in a browser.
